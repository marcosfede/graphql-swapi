# Node Star Wars GraphQL API + React Apollo Client
 

## Tech

 -  Typescript
 -  Nodejs
 -  React
 -  Apollo Client
 -  Graphql Yoga
 -  Typeorm

## Start production server
`docker-compose -f docker-compose.prod.yml  up -d --build`

## Import data
`docker-compose -f docker-compose.prod.yml exec back yarn run import`
